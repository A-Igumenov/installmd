# installmd

Скрипт установки MajorDoMo на OrangePi, Asus Tinker Board, RaspberryPi, Cubieboard, NanoPi, BananaPi и т.п.

Установка.

sudo su

apt-get update

apt-get upgrade 

rm ./installmd

wget https://raw.githubusercontent.com/immortalserg/installmd/master/installmd 

chmod +x ./installmd

./installmd
